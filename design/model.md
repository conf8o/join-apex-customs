# えぺカスタムぼ！のモデル

## 特性

- 検索性（BFのサーバー的な）
    - カスタムの検索
	    - タイトル部分一致
	    - 概要部分一致
	    - 開催予定日
        - モード
	    - マップ名部分一致
	    - マップ
	    - 参加制限
	    - エントリー人数
	    - 確定人数
        - ランク
        - 主催者
        - 参加者
- ログイン機能

## モデル

### ユーザー

- ID
- 名前
- 連絡先
- 自己紹介

### 募集

- ID
- タイトル
- 概要
- 開催予定日
- マップ
- キーマウ｜PAD｜PS5｜制限なし
- 最低ランク
- 最大ランク
- 主催者ID
- スタッフIDs
- 作成日時
- 更新日時

### 応募

- ID
- 募集ID
- ユーザーID
- コメント
- 応募日時
- 更新日時
- 承認日時
